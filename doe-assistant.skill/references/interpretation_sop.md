# 통계 어휘 ↔ 물리·공정적 의미 매핑

이 매핑은 어휘 풀이의 *출발점*이다. 강제 SOP가 아니라 LLM이 어휘를 던지고 끝내지 않게 reminder하는 도구. 케이스마다 자유롭게 변형·압축·확장한다.

**두 단계 풀이 원칙**

- **통계 어휘는 (참고용 라벨)로만** 둔다. 본체는 일상어 풀이.
- **결론·다음 스텝 절은 통계 어휘 0개로 작성 가능**하도록 점검 — 동료가 그대로 의사결정에 쓸 수 있게.
- 가능하면 이번 케이스의 숫자로 풀어낸다 (일반론 회피).

각 어휘는 두 단계로 풀어진다:
- *(공정 풀이)* — 공정·재료 도메인 어휘로 한 줄
- *(일상어 풀이)* — 통계 어휘 없이 동료가 그대로 받을 수 있는 한 줄

---

## 핵심 어휘

| 어휘 | 통계적 정의 | 한 줄 풀이 템플릿 |
|---|---|---|
| **p-value** | 귀무가설 하에서 이런 차이가 우연히 나올 확률 | "→ 이 효과가 우연일 확률 X%. {p<0.01: 매우 신뢰 / p<0.05: 신뢰 가능 / p>0.05: 우연인지 진짜인지 이번 실험만으로는 모호}" |
| **F 통계량** | 효과의 분산 ÷ 잔차 분산 (신호 ÷ 잡음 비율) | "→ 신호 강도가 잡음 대비 약 X배" |
| **η² (eta squared)** | 전체 변동 중 이 factor가 설명하는 비율 | "→ response 변동의 X%가 이 factor에서 옴 — 비교: {다른 factor}는 Y%" |
| **Cohen's f** | 효과 크기의 또 다른 척도 | "→ 0.1 작음 / 0.25 중간 / 0.4 큼. 이번엔 X로 {수준}" |
| **95% 신뢰구간** | 실제 효과가 95% 확률로 이 범위 안 | "→ 효과 크기는 [X, Y] Å/min 사이로 추정. 폭이 좁을수록 더 확실" |
| **main effect** | 다른 factor를 평균한 단독 효과 | "→ 압력을 무시하고 보면 온도만 바꿔도 식각률이 X 변함" |
| **interaction effect** | factor 조합에 따라 효과 크기가 바뀌는 정도 | "→ 압력이 낮을 때와 높을 때 온도 효과가 다르게 나옴 — 따로 떼어볼 수 없는 연동 관계" |
| **residual** | 모델이 예측한 값과 실제값의 차이 | "→ 모델이 못 잡은 잔여 변동. 패턴이 없어야 모델이 충분" |
| **fitted value** | 모델이 예측한 값 | "→ factor 조합에 대해 모델이 기대하는 response" |

---

## 설계 어휘

| 어휘 | 통계적 정의 | 한 줄 풀이 템플릿 |
|---|---|---|
| **factorial design (2^k)** | k개 factor 각각 2수준, 모든 조합 | "→ {k}개 변수를 각각 두 값으로 모든 조합 시도. 총 {2^k} run" |
| **fractional factorial** | 일부 조합만 시도해 자유도 절약 | "→ run 수를 줄이려고 일부 조합 생략. 일부 interaction 정보를 포기한 것" |
| **center points** | factor 중간값 조합을 반복 측정한 행 | "→ 곡률을 검정하고 잡음 크기를 추정하는 보조 실험" |
| **blocking** | 시간·로트 같은 외란을 분리해 모델에 넣은 것 | "→ 외부 변수의 영향을 결과에서 제거하는 장치" |
| **randomization** | 실행 순서를 무작위화 | "→ 시간 순서·드리프트 같은 숨은 효과가 결과에 새는 걸 막는 장치" |
| **lack of fit** | 모델이 못 잡은 곡률 검정 | "→ 직선 모델로 충분한지, 아니면 곡선 모델이 필요한지 판단" |

---

## 다음 설계 어휘

| 어휘 | 정의 | 한 줄 풀이 템플릿 |
|---|---|---|
| **RSM (반응표면법)** | 곡률을 모델링해 최적점을 찾는 설계 | "→ factorial로 방향을 잡은 뒤, 곡선 안쪽의 최적점을 찾을 때 사용" |
| **Box-Behnken** | 코너 없이 효율적인 RSM 설계 | "→ 극단 조건 회피하면서 곡률 추정 — 안전한 RSM" |
| **CCD (Central Composite)** | factorial + axial + center를 결합한 RSM | "→ 기존 factorial 데이터 재활용 가능한 RSM" |
| **Plackett-Burman** | 많은 factor를 적은 run으로 스크리닝 | "→ factor가 많을 때 어느 게 진짜 영향력 있는지 빠르게 거르는 도구" |
| **Taguchi** | 잡음에 강건한 설계 찾기 | "→ 평균뿐 아니라 분산도 같이 줄이는 설계 — 양산 robustness" |

## 어휘 풀이 작성 예시

**나쁜 예 (어휘 단독)**
> 온도의 main effect는 +120 Å/min, F=45.2, p=0.0003, η²=0.65로 매우 유의함.

**좋은 예 (어휘 + 풀이)**
> 온도 main effect: **+120 Å/min** (150°C → 200°C로 올리면 식각률이 평균 120 Å/min 증가).
> 신호 강도: F=45.2 → 잡음 대비 약 45배 강함. 우연일 확률은 0.03%.
> 효과 크기: η²=0.65 → 식각률 변동의 65%가 온도에서 옴. 가장 dominant factor.

**더 좋은 예 (이전 컨텍스트 인용)**
> 온도 main effect: **+120 Å/min**. 지난번 압력 영역 좁힘 실험에선 온도 효과가 +85였는데 이번엔 더 커졌네요 — 압력 영역을 좁히면서 온도의 영향력이 더 명확해진 것으로 보입니다.

## 일상어 풀이 — 풀이가 본체, 통계 근거는 함께

이 표는 결론·다음 스텝 절에서 *일상어 풀이*를 본체로 가져갈 때의 한 줄 형태. **통계 어휘 자체는 빼지 않는다** — 풀이 직후 *(통계적 근거: …)* 형태로 함께 박는다. 결론은 두 용도를 동시에 만족해야 한다.

- **의사결정용**: 동료가 풀이만 봐도 그대로 회의·이메일·다음 실험 결정에 쓸 수 있게
- **근거 자료용**: 나중에 *"이 결론이 통계적으로 어디서 왔는가"* 점검·인용 가능하게

### 결합 형식 — 풀이 + 근거 묶어 쓰는 패턴

```
- 일상어 풀이 본문 (한두 줄, 어휘 없이도 이해됨)
  *(통계적 근거: 어휘·수치 — 어떤 통계 결과가 이 풀이를 뒷받침하는가)*
```

예시 1 — 효과 강도

```
- 온도가 결과를 압도적으로 좌우합니다. 결과가 흔들린 이유의 절반 가까이가
  온도 한 변수에서 옴.
  *(통계적 근거: ANOVA F=52.1, p<0.001, η²=0.42. 우연으로 이 차이가
  나올 가능성은 천 번 중 한 번 미만.)*
```

예시 2 — 곡선 반응 확인

```
- 곡선 반응이 명확히 잡혔습니다. 1차 라운드의 의심이 확인됨.
  *(통계적 근거: 온도·압력 2차항 모두 ANOVA 유의 (p=0.001 / p=0.012),
  lack-of-fit 통과(F=0.9, p=0.45)로 RSM 모델이 영역 안에서 충분.)*
```

예시 3 — 약한 진단 위반

```
- 우리가 모르는 변수가 결과에 영향 주고 있을 가능성이 보입니다. 양산
  결정 전 점검 필요.
  *(통계적 근거: Residual vs fitted에서 줄무늬·계단 패턴, Durbin-Watson
  d=1.42로 잔차 자기상관 약하게 잡힘. 다른 진단 3종은 통과.)*
```

이 형태가 동료에게는 풀이만 읽혀 결정 자료로 쓰이고, 다른 검토자에게는 통계 근거가 추적 가능한 자료로 작동한다.

### 일상어 풀이 어휘집

| 어휘 (참고용 라벨) | 일상어 풀이 (어휘 0개) |
|---|---|
| p<0.001 | "이런 차이가 우연으로 나올 가능성은 거의 없음 — 천 번 중 한 번 미만" |
| p<0.05 | "우연으로 보긴 어려운 차이" |
| p>0.05 | "우연인지 진짜인지 이번 데이터만으론 가르기 애매" |
| F=45.2 | "이 변수의 영향이 압도적" / "효과가 매우 또렷함" |
| F=12.3 | "효과가 명확함" |
| F=5.4 | "효과가 보이긴 하지만 강하진 않음" |
| η²=0.65 | "결과가 흔들린 이유의 절반 이상이 이 한 변수 때문" |
| η²=0.18 | "결과 흔들림의 약 5분의 1을 설명" |
| η²<0.05 | "이 변수의 영향력은 거의 없음" |
| 95% CI [105, 135] | "이 효과의 진짜 크기는 거의 확실히 105~135 사이 어딘가" |
| main effect | "그 변수만 봤을 때의 단독 효과" |
| interaction | "두 변수가 함께 만들어내는 추가 효과" |
| 잔차 (residual) | "모델이 예측하지 못한 나머지 흔들림" |
| lack-of-fit 위반 | "직선만으론 부족, 곡선 반응이 있음" |
| Levene 위반 | "조건마다 결과 흔들림 크기가 다름" |
| Shapiro 위반 | "결과 흩어진 모양이 좀 비뚤어짐" |
| Residual 곡선 패턴 | "예측이 빗나가는 정도가 영역마다 달라 — 모델이 부족하다는 신호" |
| Residual 줄무늬 | "우리가 모르는 변수가 결과에 영향 주는 신호" |
| Residual 깔때기 | "큰 값일수록 결과 흔들림이 더 커짐" |
| factorial 2³ | "변수 3개를 각각 두 값으로 모든 조합 시도, 8회" |
| center points | "범위 가운데 값에서 반복 측정 — 곡선 검출 보조" |
| RSM | "안전한 범위 안에서 곡선 반응까지 잡는 추가 실험" |
| Box-Behnken | "극단 조건은 피하면서 곡선 잡는 RSM 한 종류" |
| CCD (Central Composite) | "기존 factorial 데이터 재활용해 곡선 잡는 RSM" |
| Plackett-Burman | "어떤 변수가 진짜 영향력 있는지 빠르게 거르는 실험" |
| Taguchi | "결과가 흔들리는 정도까지 줄이는 양산용 설계" |
| interior maximum | "최적값이 실험 범위의 가운데쯤에 있음" |

이 표는 결론·다음 스텝 절에서 *풀이가 어휘 없이도 의미가 살아남는지* 점검하는 도구. 살아남으면 풀이 옆에 *(통계적 근거: …)* 라벨을 박아 의사결정·근거 점검 두 용도를 함께 만족시킨다. **어휘를 빼지 않는다 — 본체와 라벨로 분리할 뿐.** LLM은 케이스마다 자유롭게 더 좋은 풀이·근거 표현을 만들어도 좋다 — 표는 출발점일 뿐.

## 주의 — 단정과 보수의 톤 전환 어휘

진단이 약할 때 결론 어조를 바꿀 때 쓰는 한국어 톤 전환구.

| 진단 강도 | 톤 전환구 |
|---|---|
| 통과 | "X입니다", "X로 나왔습니다", "X로 보입니다" |
| 약함 | "X로 보이지만, 잔차 진단이 약해 절대 수치는 잠정입니다", "방향성은 X이고 정확한 수치는 다음 실험으로 확인 권장" |
| 강한 위반 | "이번 라운드는 X 방향성을 확인한 탐색으로 보고", "결론보다 가설 단계로 보는 게 안전" |

어조 전환구를 사용해 결론을 *약화*하되, 결론 절 자체는 항상 적는다.
